GALLERY

Galleries are a beautiful way to see, show and administrate some pictures. This page is a little demo to show you how easy is configure an elegant and tiny gallery.

prettyPhoto is a great javascript and works perfectly with this kind of gallery. In minutes you have a responsive and beautiful portfolio page. Please note that if you have a minimal knowledge of CSS and HTML you can easily add some other features like a sortable option menu. The whole idea is that you have the basic tools to work with.

Galleries are not an essencial part of an administration panel, but certainly they approach new functionalities and options to developers seeking to create new panels for their projects. I'll be adding more features to this theme in some days, so I'll be really happy to listen your ideas for new tweaks and parts for this page and the whole template.
